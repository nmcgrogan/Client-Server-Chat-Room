<tr><td> <em>Response:</em> <p>The sendMessage function in the Room class is called when a message is<br>sent from a client to the server. This function first determines whether the<br>message begins with the @ character, which indicates a whisper command. If the<br>message starts with @, the sendMessage method delegated its processing to the processWhisper<br>method and returned immediately to guarantee that the whisper message was not broadcast<br>to all clients in the room.The message is separated into two pieces inside<br>the processWhisper method: the recipient&#39;s username (which comes after the @ symbol) and<br>the actual message to be whispered. This is accomplished by calling the split<br>method on the receiving message string and specifying a space character as the<br>delimiter. The processWhisper method then calls findClientByUsername to go over the list of<br>connected clients in the room and match the username to locate the associated<br>ServerThread object for the receiver.If the recipient is found, a private message is<br>sent to the ServerThread of the receiver. This is accomplished by invoking the<br>sendMessage method on the recipient&#39;s ServerThread instance, which sends the message only to<br>that client. In addition, the sender receives a confirmation message stating that the<br>whisper was sent.<br></p><br></td></tr>
